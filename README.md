
### Project Setup

#### Prerequisite
    - Node.js
    - NativeScript CLI
    - Android SDK for Android

  Follow this [link](http://docs.nativescript.org/start/quick-setup) for setting up above prerequisite.

#### Steps for setup
    1) Navigate to project directory in cmd/terminal
    2) tns platform add android
    3) npm i
    4) tns run android
